# CosmicX — Space Mission Dashboard

A real-time space mission dashboard built with React, TypeScript, and Vite. Tracks the ISS live, monitors near-Earth asteroids, visualizes solar system dynamics, and delivers ambient space immersion — all powered by NASA APIs.

**Live Site:** [akshatbehera.github.io/space-mission-dashboard](https://akshatbehera.github.io/space-mission-dashboard/)

## Features

- **ISS Live Tracking** — Real-time position on an interactive Leaflet map with latitude, longitude, altitude & velocity
- **24/7 ISS Camera Feed** — Embedded YouTube livestream from the International Space Station
- **Asteroid Monitoring** — Near-Earth object data with distance, speed & hazard assessment
- **Sunrise & Sunset Calculator** — Local times for 10 major cities with animated sun arc
- **Earth–Sun Simulation** — Canvas animation with orbiting Earth, Moon, and sun corona effects
- **Solar System Simulation** — All 8 planets with gradient rendering, glow effects & Saturn's ring
- **Space on X** — Live feeds from SpaceX, NASA Solar System & Space Alerts (with fallback cards)
- **Ambient Space Audio** — Auto-playing background music with play/pause & mute controls
- **Space Trivia Quiz** — Test your knowledge with instant feedback
- **Scrolling Ticker** — Space facts and trivia running across the screen
- **Background Slideshow** — Cinematic space imagery fading behind the UI
- **Extreme Responsiveness** — Fully optimized for desktop, tablet & mobile
